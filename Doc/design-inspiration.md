## TECMAHに合うデザインの方向性

### 推奨デザインスタイル

#### 1. **ライト・ミニマル + アクセントカラー**
- **背景**: 白または明るいグレー（`#F8FAFC`, `#FFFFFF`）
- **アクセント**: TECMAHグリーン（`#00d4aa`）を効果的に使用
- **特徴**: クリーンで信頼感のある印象

#### 2. **グラデーション + ホワイトベース**
- **背景**: 白ベースにグラデーションオーバーレイ
- **カラー**: 明るいブルー、グリーン、パープルのグラデーション
- **特徴**: モダンで技術的な印象

#### 3. **ニューモーフィズム風**
- **背景**: 明るいグレー（`#E5E7EB`）
- **カード**: ソフトシャドウで浮き上がる効果
- **特徴**: 柔らかく親しみやすい印象

## Awwwardsでの具体的な検索方法

### 検索手順

1. **トップページからカテゴリを選択**
   - "By Category" → "Business & Corporate"
   - "By Category" → "Design Agencies"

2. **フィルター機能を活用**
   - 左サイドバーのフィルターで「Light」や「Minimal」を選択
   - 色でフィルタリング（白、明るい色調）

3. **コレクションを確認**
   - "Collections" → "AI Powered Web Projects"
   - "Trending" → "Portfolio Websites"

4. **Sites of the Day/Monthをチェック**
   - 最新トレンドのデザインを確認
   - 特に明るいカラースキームのサイトをピックアップ
