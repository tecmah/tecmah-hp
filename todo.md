# TECMAH サイト改修タスク

## フェーズ1: コンセプト & デザイン
- [ ] **デザインコンセプトの確定**
    - [ ] 3つの提案（モダンミニマル、ハイテック、プレミアム）から1つを選択
    - [ ] 最終的なカラーパレットとタイポグラフィを定義
- [ ] **アセット準備**
    - [ ] ヒーロービジュアルの作成/調達（3D抽象画または高品質な写真）
    - [ ] サービス用アイコンの準備（GenAI、PoCなど）
    - [ ] 全セクションのコピーテキストを収集

## フェーズ2: 開発環境
- [ ] **Astroプロジェクトのセットアップ**
    - [ ] AstroとTailwind CSSの統合による初期化
    - [ ] `src/styles/global.css`にベーススタイルを設定
    - [ ] フォント読み込み設定（Google Fontsまたはローカル）

## フェーズ3: コンポーネント実装
- [ ] **レイアウトコンポーネント**
    - [ ] `Header.astro`: ブラー効果付きスティッキーナビゲーション
    - [ ] `Footer.astro`: サイトマップ、SNSリンク、コピーライト
- [ ] **ページセクション**
    - [ ] `Hero.astro`: 没入感のあるエントランスアニメーション
    - [ ] `Services.astro`: サービス提供内容のグリッドレイアウト
    - [ ] `Trust.astro`: 実績のカウンターアニメーション
    - [ ] `Mission.astro`: パララックスを用いたストーリーテリングセクション
    - [ ] `CTA.astro`: 高コンバージョンな問い合わせフォーム/ボタン

## フェーズ4: アニメーション & 仕上げ
- [ ] **スクロールアニメーション**
    - [ ] スクロールトリガーのフェードイン実装（Intersection ObserverまたはMotion One使用）
    - [ ] 背景要素へのパララックス効果追加
- [ ] **マイクロインタラクション**
    - [ ] ボタンとカードのホバー状態
    - [ ] アンカーリンクのスムーススクロール

## フェーズ5: 最適化 & ローンチ
- [ ] **パフォーマンス**
    - [ ] 画像の最適化（WebP/AVIF）
    - [ ] CSS/JSの最小化
- [ ] **SEO**
    - [ ] メタタグの追加（タイトル、説明、OG画像）
    - [ ] sitemap.xmlの生成
- [ ] **デプロイ**
    - [ ] Vercel/Netlify/Cloudflare Pagesへのデプロイ
